Я знаю такі GIT команди

  • git clone адрес репозиторія - клонувати репозиторій на локальному комп'ютері
  • git commit -m 'початкова фіксація' фіксує зміни в локальному репозиторії
  • git push -u origin master - тільки перший раз відправка змін у віддалений репозиторій
  • git push - усі наступні випадки відновлення змін у віддаленому репозиторії
  • git pull - завантажується актуальна версія віддаленого репозиторію
